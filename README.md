# Онлайн магазин Auxilium Shop

## Стек и платформа
- **Стек:** React Native (Expo SDK 56)
- **Целевая платформа:** Android
- **Язык:** JavaScript
- **Сборка:** Development Build (expo-dev-client)
- **Система контроля версий:** Git + GitHub

## Краткое описание приложения
Auxilium Shop - приложение на Android, которое представляет онлайн-магазин для покупки различных товаров

## Сборка проекта

### Требования к окружению
- Node.js **20 LTS** или **22 LTS**
- npm (устанавливается вместе с Node.js)
- JDK 17
- Эмулятор или физическое устройство

### Шаги для клонирования и запуска

1. **Клонировать репозиторий и перейти в папку проекта**

2. **Установить зависимости**
   ```bash
   npm install
   ```
3. **Установить и настроить expo-dev-client**
   ```bash
   npx expo install expo-dev-client
   ```

4. Собрать development‑версию приложения (первый раз)
   ```bash
   npx expo run:android
   ```
